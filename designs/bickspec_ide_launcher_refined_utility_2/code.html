BickSpec

A specification language for finance

A structured DSL for financial and economic engineering. Designed to formalize financial logic with clarity, consistency, and machine readability.

Code Preview more_horiz 
spec PortfolioAnalysis {
  input assets: Array<Asset>;
  input horizon: Duration = "1y";

  // Compute weighted CAGR
  calculate expected_return = 
    assets.sum(a => a.weight * a.cagr);
  
  validate total_weight {
    assets.sum(a => a.weight) == 1.0;
  }
}
